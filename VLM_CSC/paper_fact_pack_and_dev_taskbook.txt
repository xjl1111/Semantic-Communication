1. 论文任务目标
【论文明确写出】
论文提出的是 Visual Language Model-based Cross-modal Semantic Communication system（VLM-CSC）。
它要解决传统图像语义通信在动态环境下的三个问题：
(1) 图像像素直接传输的信息密度低；
(2) 动态场景下持续学习会产生灾难性遗忘；
(3) 训练与测试 SNR 不匹配时鲁棒性下降。

论文提出三项核心设计：
- CKB：跨模态知识库。发送端用 BLIP 做 image-to-text，接收端用 SD 做 text-to-image。
- MED：Memory-Assisted Encoder and Decoder，通过 STM/LTM 缓解持续学习遗忘。
- NAM：Noise Attention Module，利用 SNR 自适应调节特征。

2. 系统总体组成
【论文明确写出】
系统由发送端、物理信道、接收端三部分组成。
输入图像定义为：
\[
x \in \mathbb{R}^{H \times W \times C}
\]

发送端整体映射：
\[
y = C_{\beta}(S_{\alpha}(K_{\theta}(x), \mu), \mu)
\]

接收端整体映射：
\[
\hat{x} = K^{-1}_{\theta'} \big( S^{-1}_{\delta}(C^{-1}_{\gamma}(\hat{y}, \mu), \mu) \big)
\]

信道模型：
\[
\hat{y} = h \cdot y + n
\]
其中，h 为信道增益，n 为 AWGN 噪声。

3. 已确认的数据流
【论文明确写出】
完整主链路为：
image x
-> BLIP-based CKB
-> text s
-> semantic encoder
-> channel encoder
-> channel
-> channel decoder
-> semantic decoder
-> recovered text \hat{s}
-> SD-based CKB
-> reconstructed image \hat{x}

结论：系统主传输对象是文本语义，而不是直接传输图像像素。

以下两个模块是预训练 VLM，不需要为 CSC 系统重新训练：
- BLIP-based CKB
- SD-based CKB

【工程含义】

- MED 机制相关逻辑
而不是重新训练 BLIP 或 SD。

5. BLIP-based CKB
【论文明确写出】
发送端使用 BLIP 做图像到文本语义提取。
结构包括：
- image encoder
- image-grounded text decoder

图像编码器：
- 基于 ViT
- 图像先切为 patch sequence x_p
- 多层 encoder layer，每层包含 MSA 与 FF

第一层 MSA 输出：
\[
m_{\mathrm{msa},1} = \mathrm{MSA}(\mathrm{LN}(x_p)) + x_p
\]

第一层 FF 输出：
\[
m_{\mathrm{ff},1} = \mathrm{GeLU}(W_{b,f}\cdot \mathrm{LN}(m_{\mathrm{msa},1}) + b_{b,f}) + m_{\mathrm{msa},1}
\]

第 L 层输出：
\[
m_L = \mathrm{LN}(m_{\mathrm{ff},L})
\]

文本解码器：
- 采用 BERT 风格结构
- 每层包含 Causal Self-Attention、Cross Attention、FF
- 初始 token 为 "[Decoder]"

第一层：
\[
k_{\mathrm{csa},1} = \mathrm{CSA}(\mathrm{LN}(D_0)) + D_0
\]
\[
k_{\mathrm{ca},1} = \mathrm{CA}(\mathrm{LN}(k_{\mathrm{csa},1}), m_L) + k_{\mathrm{csa},1}
\]
\[
k_{\mathrm{ff},1} = \mathrm{ReLU}(W'_{b,f}\cdot \mathrm{LN}(k_{\mathrm{ca},1}) + b'_{b,f}) + k_{\mathrm{ca},1}
\]

最后通过线性投影和 softmax 逐步生成文本描述 s。

已确认输入输出：
- 输入：图像 x \in \mathbb{R}^{H \times W \times C}
- 输出：文本描述 s（token sequence / caption）

6. SD-based CKB
【论文明确写出】
接收端使用 Stable Diffusion 做文本到图像重建。
组件包括：
- text encoder
- feature generator
- image decoder
其中 feature generator 核心为 denoising U-Net。

text encoder：
- 把恢复文本 \hat{s} 映射为固定维语义向量
- 由多层 stack 组成，每层包含 MSA 与 FF
- 先进行 word embedding，再迭代编码得到 textual feature vectors

扩散去噪迭代：
\[
Z_{t-1} =
\frac{1}{\sqrt{\alpha_t}}
\left(
Z_t -
\frac{1-\alpha_t}{\sqrt{1-\bar{\alpha}_t}} f_{\theta}(Z_t,t,d)
\right)
+ \sigma_t Y
\]
其中：
- Z_t：时刻 t 的图像特征向量
- \alpha_t：前向扩散方差超参数
- \bar{\alpha}_t = \prod_{i=1}^{t} \alpha_i
- f_{\theta}：预训练 noise prediction U-Net
- d：文本语义向量
- \sigma_t = \sqrt{1-\bar{\alpha}_t}
- Y \sim \mathcal{N}(0, I)

image decoder：
- 在压缩语义空间完成去噪
- 最后用 VAE decoder 从 latent 恢复到像素空间，得到 \hat{x}

已确认输入输出：
- 输入：恢复文本 \hat{s}
- 输出：重建图像 \hat{x}

9. MED 机制
【论文明确写出】
MED 使用：
- STM
- LTM

记号：
\[
M_{\mathrm{stm}} = \{s_i^{\mathrm{stm}}\}_{i=1}^{n_{\mathrm{stm}}}, \quad
M_{\mathrm{ltm}} = \{s_j^{\mathrm{ltm}}\}_{j=1}^{n_{\mathrm{ltm}}}
\]

STM 满时，从 STM 中选择代表样本转入 LTM。
最大 STM 容量记为 n_{stm}^{max}。

RBF 相似度：
\[
\mathrm{RBF}(s_i^{\mathrm{stm}}, s_j^{\mathrm{ltm}})
=
\exp\left(
-\frac{\|v_i^{\mathrm{stm}} - v_j^{\mathrm{ltm}}\|^2}{2\tau^2}
\right)
\]
其中：
- v_i^{stm}, v_j^{ltm} 为 semantic encoder 提取的特征向量
- \tau = 10

平均相似度：
\[
R(s_i^{\mathrm{stm}})
=
\frac{1}{n_{\mathrm{ltm}}}
\sum_{k=1}^{n_{\mathrm{ltm}}}
\mathrm{RBF}(s_i^{\mathrm{stm}}, s_k^{\mathrm{ltm}})
\]

样本转移规则：
\[
R(s_i^{\mathrm{stm}}) > \lambda
\Rightarrow
M_{\mathrm{ltm}} = M_{\mathrm{ltm}} \cup s_i^{\mathrm{stm}}
\]

实验中明确给出：
- STM 最大样本数 = 500
- sample selection threshold = 0.05

10. NAM 机制
【论文明确写出】
NAM 输入：
- SNR 值 r
- 中间特征向量 G
输出：
- 校准特征向量 A

SNR projection：
\[
v' = \mathrm{ReLU}(W_{n2}\cdot \mathrm{ReLU}(W_{n1}\cdot r + b_{n1}) + b_{n2})
\]
\[
v = \mathrm{Sigmoid}(W_{n3}\cdot v' + b_{n3})
\]

feature scaling：
\[
K = \mathrm{Sigmoid}(e \cdot v)
\]
\[
e = W_{n4}\cdot G + b_{n4}
\]
\[
A_i = K_i \cdot G_i
\]

NAM 的 4 层 FF 神经元数：
56, 128, 56, 56

11. 训练目标与损失
【论文明确写出】
文本恢复损失为 CE：
\[
L_{\mathrm{CE}}(s,\hat{s})
=
-\sum_{l=1}^{L}
q(w_l)\log p(w_i)
+
(1-q(w_l))\log(1-p(w_i))
\]
说明：论文记号在此有不统一，但核心意义是让恢复文本 \hat{s} 的词分布逼近原始文本 s 的词分布。

整体优化目标：
\[
(\alpha^*, \beta^*, \delta^*, \gamma^*)
=
\arg\min_{\alpha,\beta,\delta,\gamma}
\mathbb{E}_{p(\mu)}\mathbb{E}_{p(s,\hat{s})}
\left[L_{\mathrm{CE}}(s,\hat{s})\right]
\]

论文还明确写出：
第一阶段训练 channel encoder / decoder + NAM 的目标是 minimizing the mutual information。

14. 评估指标
【论文明确写出】
系统从图像层与文本层两个角度评估。

图像层：SSQ
\[
\mathrm{SSQ} = \frac{ST(\hat{S})}{ST(S)}
\]
其中 ST(·) 在本文中表示分类准确率。

文本层：BLEU
\[
\log \mathrm{BLEU}
=
\min\left(1-\frac{l_{\hat{s}}}{l_s},0\right)
+
\sum_{n=1}^{N} u_n \log p_n
\]
\[
p_n
=
\frac{\sum_k \min(C_k(\hat{s}), C_k(s))}
{\sum_k \min(C_k(\hat{s}))}
\]
