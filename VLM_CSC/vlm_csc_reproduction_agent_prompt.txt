A. 系统总流程（论文明确写出）
- 发射端输入图像 \(x \in \mathbb{R}^{H \times W \times C}\)。
- 发射端由三部分组成：CKB、semantic encoder、channel encoder。
- 发送符号表示为：
  \[
  y = C_\beta\big(S_\alpha(K_\theta(x), \mu), \mu\big)
  \]
- 信道模型：
  \[
  \hat{y} = h \cdot y + n
  \]
  其中 \(n\) 为 AWGN。
- 接收端由三部分组成：channel decoder、semantic decoder、cross-modal knowledge base for reconstruction。
- 重建图像表示为：
  \[
  \hat{x} = K_{\theta'}^{-1}\Big(S_\delta^{-1}(C_\gamma^{-1}(\hat{y}, \mu), \mu)\Big)
  \]
- 文本一致性损失采用交叉熵：
  \[
  L_{CE}(s,\hat{s}) = -\sum_{l=1}^{L} q(w_l)\log p(w_i) + (1-q(w_l))\log(1-p(w_i))
  \]
- 目标函数：
  \[
  (\alpha^*,\beta^*,\delta^*,\gamma^*) = \arg\min_{\alpha,\beta,\delta,\gamma} \mathbb{E}_{p(\mu)}\mathbb{E}_{p(s,\hat{s})}[L_{CE}(s,\hat{s})]
  \]

B. VLM-CSC 的 3 个核心贡献（论文明确写出）
- CKB：发射端用 BLIP 从图像提取高信息密度文本语义；接收端用 SD 根据文本重建图像。
- MED：STM + LTM 混合记忆，缓解 continual learning 中的 catastrophic forgetting。
- NAM：基于 SNR 的注意力模块，动态调整 semantic coding 与 channel coding 的权重。

C. 发射端 CKB（BLIP-based）结构（论文明确写出）
- 使用 BLIP 的 image encoder + image-grounded text decoder。
- image encoder 基于 ViT，将图像分成 patch，经过 MSA + FF 子层。
- 第 1 个 image encoder layer 的写法：
  \[
  m_{msa,1} = MSA(LN(x_p)) + x_p
  \]
  \[
  m_{ff,1} = GeLU(W_{b,f} \cdot LN(m_{msa,1}) + b_{b,f}) + m_{msa,1}
  \]
- 第 \(L\) 个 image encoder layer 输出：
  \[
  m_L = LN(m_{ff,L})
  \]
- text decoder 由 CSA、CA、FF 组成。
- 第 1 个 text decoder layer：
  \[
  k_{csa,1} = CSA(LN(D_0)) + D_0
  \]
  \[
  k_{ca,1} = CA(LN(k_{csa,1}), m_L) + k_{csa,1}
  \]
  \[
  k_{ff,1} = ReLU(W'_{b,f} \cdot LN(k_{ca,1}) + b'_{b,f}) + k_{ca,1}
  \]
- 其最终输出是图像文本描述 \(s\)。

D. 接收端 CKB（SD-based）结构（论文明确写出）
- 使用 SD 的 text encoder、denoising U-Net、image decoder。
- text encoder 输出固定维度语义向量，作为 diffusion 过程条件。
- diffusion generator 的迭代公式：
  \[
  Z_{t-1} = \frac{1}{\sqrt{\alpha_t}}\left(Z_t - \frac{1-\alpha_t}{\sqrt{1-\bar{\alpha}_t}} f_\theta(Z_t, t, d)\right) + \sigma_t Y
  \]
- image decoder 使用 VAE decoder，将语义空间特征映射回像素空间。

E. MED（论文明确写出）
- 同时维护 STM 和 LTM。
- 当 STM 满时，计算 STM 与 LTM 的核距离，选择代表样本转移至 LTM。
- RBF 核：
  \[
  RBF(s_i^{stm}, s_j^{ltm}) = \exp\left(-\frac{\|v_i^{stm}-v_j^{ltm}\|^2}{2\tau^2}\right)
  \]
- 论文明确给出 \(\tau = 10\)。
- 平均相似度：
  \[
  R(s_i^{stm}) = \frac{1}{n_{ltm}}\sum_{k=1}^{n_{ltm}} RBF(s_i^{stm}, s_k^{ltm})
  \]
- 转移规则：
  \[
  R(s_i^{stm}) > \lambda \Rightarrow M_{ltm} = M_{ltm} \cup s_i^{stm}
  \]
- 论文明确给出：STM 最大样本数 500；sample selection threshold 为 0.05。

F. NAM（论文明确写出）
- 基于 SNR 的 attention/gating 模块。
- SNR projection：3 层 FF，前两层 ReLU，第三层 Sigmoid。
  \[
  v' = ReLU(W_{n2}\cdot ReLU(W_{n1}\cdot r + b_{n1}) + b_{n2})
  \]
  \[
  v = Sigmoid(W_{n3}\cdot v' + b_{n3})
  \]
- feature scaling：
  \[
  K = Sigmoid(e \cdot v)
  \]
  \[
  e = W_{n4}\cdot G + b_{n4}
  \]
  \[
  A_i = K_i \cdot G_i
  \]
- 论文明确给出：NAM 4 个 FF 层神经元数为 56, 128, 56, 56。

G. 训练/实验中明确写出的模型设置（论文明确写出）
- semantic encoder：3 个 transformer encoder layers，与 NAM 交替。
- 每个 transformer encoder layer：8 heads，feature dimension = 128。
- channel encoder：2 个 FF hidden layers，与 NAM 交替。
- 2 个 hidden 层神经元数：256、128。
- semantic decoder / channel decoder：结构与 encoder 对称、相反。
- BLIP 预训练参数量：129MB。
- SD 预训练参数量：1.99GB。
- STM 最大样本数：500。
- sample selection threshold：0.05。
- 实验环境：Windows Server 2016, Python 3.8, PyTorch 1.8.0, CUDA 11.6, CPU Intel Xeon Silver 4210R 2.40GHz, GPU Tesla T4。

H. 评估指标（论文明确写出）
- Image-level：SSQ
  \[
  SSQ = \frac{ST(\hat{S})}{ST(S)}
  \]
  其中 \(ST(\cdot)\) 在本文中是 classification accuracy。
- Text-level：BLEU
  \[
  \log BLEU = \min(1 - \frac{\hat{l}_s}{l_s}, 0) + \sum_{n=1}^{N} u_n \log p_n
  \]
  \[
  p_n = \frac{\sum_k \min(C_k(\hat{s}), C_k(s))}{\sum_k \min(C_k(\hat{s}))}
  \]

====================================
六、训练流程必须拆成三阶段，并可交替迭代
====================================
按论文训练逻辑实现，不要改成一次性端到端黑盒训练。

1. Stage A：训练 channel encoder / decoder + NAM
- 冻结 BLIP、SD、semantic encoder、semantic decoder。
- 输入来自 semantic encoder 的特征，经过 channel encoder -> channel -> channel decoder。
- 优化目标：论文说是 minimization of mutual information / 消除噪声影响；采用你在 assumptions.md 记录的 proxy loss。
- 训练完成后冻结 channel encoder / decoder + NAM 参数。

2. Stage B：训练 semantic encoder / decoder + NAM
- 冻结 BLIP、SD、channel encoder、channel decoder。
- 训练 semantic encoder / decoder，在信道存在下恢复文本。
- 损失使用 CE。
- 训练完成后冻结 semantic encoder / decoder + NAM。

3. Stage C：crossover-based iterative training
- 交替执行 Stage A 与 Stage B，直到验证集不再提升。
- 每轮交替都保存 checkpoint。
- `trainers/trainer_joint.py` 实现早停、最佳模型选择、日志记录。

4. continual learning / MED 接入方式
- Fig.8、Fig.9、Fig.10 训练时，训练迭代器应支持 MED。
- 每个 step 或每个 epoch 后：
  - 把当前样本加入 STM
  - 满容量时做 RBF relevance evaluation
  - 按规则转移至 LTM
  - 训练 batch 由 STM/LTM 按比例混合抽样
- 论文未给出混合比例，这是“为复现做的合理实现选择”；默认 1:1，并记录。

1. 复现 Fig.7：VLM-based KB 比较
目标：比较 BLIP / LEMON / RAM 作为 sender-side KB 的 SSQ。

实现要求：

2. 复现 Fig.8：MED ablation
目标：比较有无 MED 在 continual learning 下的 BLEU 退化情况。

实现要求：

- channel：Rayleigh（论文明确写出）。

  - 有 MED 的 BLEU-1
  - 有 MED 的 BLEU-2

验收标准：

3. 复现 Fig.9：NAM ablation
目标：验证 NAM 缓解 train-test SNR mismatch 的能力。

实现要求：

- NAM-off：分别在固定训练 SNR = {0,2,4,8} dB 下训练 4 个模型
- 测试时：`SNR_test = 0,1,2,...,10 dB`
- 指标：BLEU
- 作图：x 轴 `SNR_test(dB)`，y 轴 BLEU

验收标准：
- NAM-on 曲线应整体优于各个固定 SNR 训练的无 NAM 模型。
- 若不成立，先检查 NAM 是否真正插入 semantic/channel encoder 和 decoder 的每层后。

4. 复现 Fig.10：与 JSCC/WITT 对比
目标：比较 VLM-CSC、JSCC、WITT。

实现要求：
- dataset：CATSvsDOGS。
- 指标至少包含：
  - SSQ / classification accuracy
  - compression ratio
  - trainable parameters
  - semantic alignment 可视化
- baseline：
  - JSCC：CNN-based
  - WITT：ViT-based
- 由于论文未给出 baseline 精确超参数，因此在 assumptions.md 中标记为“为复现做的合理实现选择”，但要尽量采用公开标准实现。

compression ratio 的实现要求：
- 论文未给出精确公式，因此你必须显式定义：
  \[
